# Welcome to My Blog!

This is a static blog application with claymorphic design, built with TypeScript and Express.

## Features

- **Markdown Support**: Write posts in Markdown with YAML frontmatter
- **Dark/Light Mode**: Toggle between themes with a click
- **Daily Sync**: Automatically pulls updates from your Git repository

## Quick Start

1. Clone your private blog repository to `repos/posts/`
2. Run `npm run build` to generate the site
3. Start the server with `npm start`
4. Visit `http://localhost:3000`

## Customization

Edit `site/css/theme.css` to customize the claymorphic design colors and styling.
